=== DOMINEX ===

To start the game, run .\Window\bin\Release\Window.exe application.

If you want to generate new strategies, run .\KingdomEvolution\bin\Release\KingdomEvolution.exe
Parameters cannot be combined to one string.
parameters:
-f           -- Generating fives. All fives are listed in .\AI\Provincial\Data\kingdoms\fives.txt
-h           -- Generating threes. All threes are listed in .\AI\Provincial\Data\kingdoms\threes.txt
-d           -- Generating randomly choosen tens.
-t [number]  -- Maximal degree of parallelism.
-c [number]  -- Number of the new strategies.
-s [number]  -- Number of the strategies in the fives or threes list that will be skipped.
